Styling Text

Text Styling Properties
Property Fucntion Example Code
color Changes the color of text. p {
color: black;
}
background-color Changes color of an element behind text. p {
background-color: white;
}
font-family Changes font of text. p {
font-family: Arial;
}
font-size Changes size of font p {
font-size: 72px;
}
font-weight Changes line weight of font, i.e., boldness. p {
font-weight: bold;
}
font-style Changes style of font, i.e., italics. p {
font-style: italic;
}
text-transform Transforms all text in an element, i.e., to upper or lowercase. p {
text-transform: uppercase;
}
text-decoration Decorates text, i.e., with a strikethrough. p {
text-decorate: line-through;
}
text-shadow Creates a shadow on text in an element. p {
text-shadow: 2px, 2px, 2px, red;
}
text-align Aligns text to a specified position. p {
text-align: cemter;
}
line-height Changes the vertical distance between lines of text. p {
line-height: 1.5;
}
letter-spacing Changes the amount of space between letters in a word. p {
letter-spacing: 0.5em;
}
word-spacing Changes the amount of space between words in text. p {
word-spacing: 2px;
}
